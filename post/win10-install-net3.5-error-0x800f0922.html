Win10安装.net3.5 出现错误代码0x800f0922 解决方法

2020-03-12

看到很多人在Win10/Win8上安装.net 3.5都会报这个错，然而网上的答案并不靠谱，所以笔者这里提供一个正确的打开方式。

  1. 按Win+X——程序和功能——左侧的启用或关闭功能
  2. 去掉.net 4.6的勾选，点确定，等待完成
  3. 重复第1步，勾选.net 3.5，点确定，等待完成
  4. 重复第1步，勾选.net 4.6，点确定，等待完成。
  5. 大功告成。
