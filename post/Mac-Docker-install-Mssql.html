Mac安装linux mssql

2020-03-08

为了操作方便，在mac上也可以安装mssql，需要借助docker，之前都是vmware或者visualbox安装windows再装mssql，现在这样方便了。

  1. mac安装docker

  2. 安装Kitematic,这个是可视版的，主页搜索mssql即可进行安装（这里需要翻墙）

  3. 下载好的mssql运行需要设置两个参数

    ACCEPT_EULA=Y
SA_PASSWORD=G1veMeP@ss

    其中对应的端口本机和docker都改成1433.
